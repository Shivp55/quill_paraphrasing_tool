Velocities are vector quantities with units of meters/second (m/s). An important vector quantity in physics is an object's change in velocity, which is gotten by subtracting the object's velocity at the start of some time interval from the object's velocity at the end of that time interval; changes in velocity are used to calculate acceleration vectors, which appear in Isaac Newton's Laws of Motion.
